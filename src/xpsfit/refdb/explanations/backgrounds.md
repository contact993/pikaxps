# 백그라운드 (Background)

XPS 피크 아래에는 비탄성 산란된 전자들이 만드는 백그라운드가 깔려 있습니다.
피크 면적은 "백그라운드를 뺀 나머지"이므로, **백그라운드 선택이 정량 결과를 직접 좌우합니다.**

## Linear (선형)

두 끝점을 직선으로 연결합니다.

- **언제**: 백그라운드 변화가 작은 약한 피크 (C 1s, N 1s, O 1s 등 가벼운 원소), 좁은 에너지 창
- **주의**: 전이금속 2p처럼 피크 전후 단차가 큰 영역에는 부적합 — 면적이 왜곡됩니다

## Shirley

"피크 아래 누적 면적에 비례해 계단처럼 올라가는" 백그라운드입니다. 비탄성 산란 전자는
원래 피크보다 **높은 결합에너지 쪽**에 나타나므로, 백그라운드도 고BE 쪽으로 상승합니다.
반복 계산(iterative)으로 자기일관적으로 수렴시킵니다.

- **언제**: 대부분의 금속/전이금속 영역의 기본값. XPS fitting의 사실상 표준
- **주의**: **끝점(endpoint) 위치에 민감** — 끝점을 피크 꼬리 위에 잡으면 면적이 수 % 단위로 흔들립니다.
  끝점은 피크에서 충분히 떨어진 평탄한 곳에 두고, 끝점 위치를 ±1 eV 움직여봐서 결과 민감도를 확인하세요.
- 이 프로그램은 끝점을 주변 여러 점의 평균으로 잡아 노이즈 민감도를 줄입니다 (XPSPEAK와 동일)

### BG 동시 fit (Active Shirley)

기본 Shirley는 **측정 데이터**의 누적 면적으로 백그라운드를 만들기 때문에 노이즈와
창 안의 다른 피크 구조에 민감합니다. "BG 동시 fit"을 켜면 백그라운드를 **피크 모델**에서
유도해(Sherwood 방식) 피크와 함께 최적화합니다 — fitting을 반복할수록 피크와 백그라운드가
서로 맞춰지며 수렴합니다. 일반 Shirley로 백그라운드가 이상하게 잡힐 때 켜 보세요.

## Shirley + Linear

Shirley에 선형 기울기를 더한 형태 (XPSPEAK의 "Shirley+Linear"). 시료 전체에 기울어진
배경(다른 피크의 꼬리 등)이 깔려 있을 때 사용합니다. slope=0이면 순수 Shirley와 동일합니다.

## Tougaard

비탄성 산란의 물리(universal loss function, F(T) = B·T/(C+T²)²)를 명시적으로 적분하는
가장 물리적으로 엄밀한 방법입니다.

- **언제**: 절대 정량이 중요할 때, 넓은 에너지 범위(피크 후 30 eV 이상)를 측정했을 때
- **주의**: 좁은 창에서는 부정확합니다. 일상적인 화학 상태 분석에는 Shirley로 충분한 경우가 대부분
- 파라미터: C = 1643 eV² (universal), B는 데이터 끝점에 맞춰 자동 스케일 (수동 조정 가능)

## 실무 요약

| 상황 | 추천 |
|---|---|
| 전이금속 2p (Ni, Co, Fe, ...) | Shirley |
| C 1s, N 1s, O 1s | Shirley 또는 Linear |
| 절대 정량, 넓은 측정 범위 | Tougaard |
| 기울어진 baseline 위의 피크 | Shirley+Linear |

> 백그라운드 종류를 바꾸면 면적이 달라집니다. **논문에는 어떤 백그라운드를 썼는지 반드시 명시**하세요.
