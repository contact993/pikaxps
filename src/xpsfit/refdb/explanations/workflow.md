# 권장 Fitting 절차

## 1. 준비 — 대전 보정부터

1. **C 1s region을 먼저 피크분리**: adventitious C-C 성분(~284.8 부근)을 fitting
2. **Regions 패널의 BE Shift**: "C-C→284.8" 버튼을 누르면 필요한 시프트가 자동 계산됨
   (또는 ± 값을 직접 입력) → "Shift 적용"을 누르면 **모든 region(O 1s, 금속 피크 등)이
   같은 값으로 이동**. 일부만 옮기려면 "전체 적용"을 끄고 목록에서 선택 후 적용
3. Survey 스캔으로 존재 원소 확인 — 예상 못한 원소(오염)는 fitting 전에 파악
4. 영역 선택: 피크 양쪽으로 **평탄한 구간이 보일 만큼** 넓게. 창에 이웃 피크가 침범하면
   BG 세로선을 끌어 범위를 잘라내세요 (창 밖 데이터는 흐리게 표시되고 fitting에서 제외;
   "↔ Range reset"으로 취소)

## 2. 백그라운드

1. 종류 선택 (기본: Shirley — Help의 "백그라운드" 참조)
2. 끝점을 피크 꼬리에서 충분히 떨어진 곳에 배치
3. 끝점을 ±1 eV 움직여 면적 민감도 확인 — 크게 흔들리면 영역을 넓혀 재측정 고려

## 3. 피크 모델 구성

1. **레퍼런스 DB에서 시작**: 원소 검색 → 예상 화학 상태의 가이드라인을 띄워 데이터와 대조
2. 화학적 근거가 있는 성분만 추가 (recipe 사용 권장)
3. doublet은 반드시 splitting/비율 고정으로 (Insert Doublet 기능)
4. 같은 종류 성분끼리 FWHM 링크
5. 금속 성분은 비대칭 라인섀입 (GL+Tail 또는 Doniach-Šunjić)

## 4. Fitting

1. 피크를 대략적 위치에 놓고 (드래그) **Optimise Region**
2. 수렴 후 잔차 확인: 구조적 패턴이 남아 있으면 모델 부족/과잉
3. 초기값을 바꿔 다시 fitting — 같은 답이 나와야 신뢰 가능

## 5. 검증 체크리스트 (논문 제출 전)

- [ ] FWHM이 전부 물리적 범위(0.7–2.5 eV, envelope 제외)인가
- [ ] doublet 비율/간격이 고정되어 있는가
- [ ] center가 문헌 범위 안인가 (DB 대조)
- [ ] 잔차가 평평한가
- [ ] 백그라운드 종류·범위, 라인섀입, constraint, 보정 기준을 **모두 보고**했는가
- [ ] 전이금속 multiplet(Ni 2p, Co 2p, Fe 2p, Cr 2p, Mn 2p)을 단순 1피크로 처리하지 않았는가

## 잔차(Residual) 읽는 법 — "피크가 하나 더 필요한가?"

플롯 아래의 잔차 = **측정 데이터 − 모델(피크+배경)**.

- 잔차가 0 주위에 **무작위 노이즈**로만 흩어져 있으면 → 모델 충분. 성분을 더 넣으면
  χ²는 항상 줄지만 화학적 의미는 없습니다 (과적합)
- 잔차에 **피크 폭 규모의 물결/언덕**이 남아 있으면 → 그 위치에 모델이 못 잡은 무언가가
  있다는 신호 (빠진 성분, satellite, 비대칭, 배경 문제 중 하나)

**🔍 진단 버튼**은 리뷰어가 보는 항목을 종합 감사해 ✖/⚠/✓ 보고서로 정리합니다:

| 항목 | 검사 내용 |
|---|---|
| FWHM | < 0.35 eV(노이즈 피팅) / > 3 eV(satellite 라벨 없으면) / 경계값에 붙음 |
| 레퍼런스 | 각 피크 BE를 DB의 알려진 화학종 범위와 대조, 안 맞으면 가장 가까운 상태 제시 |
| Doublet | 짝이 창 안에 있어야 하는데 없는 경우 / 풀린 짝의 실제 간격·면적비 vs 이론값 |
| 라인섀입 | 금속성 상태(DB hint)를 대칭 GL로 피팅 → 산화물 과대평가 경고 |
| Satellite | Cu(II)/Co(II)/Ni(II) 등 satellite 동반 상태인데 satellite 피크가 없으면 |
| 과적합 | 간격 < 0.4 eV 피크 쌍, 기여 < 1% 성분, 면적 0 성분 |
| **필요성** | **피크를 하나씩 빼고 재fitting(leave-one-out)** — 빼도 결과가 같으면(ΔBIC ≤ 0) "데이터가 요구하지 않는 피크"로 지적. "추가 전후가 똑같은" 피크를 정량적으로 잡아냄 |
| 대전 보정 | C 1s 영역이면 C-C가 284.8 ± 0.25 안인지 |
| 잔차 통계 | 밴드패스 z-score + 피크 시험 추가 ΔBIC (< −10 = 강한 추가 근거) |

> 통계와 물리/문헌 검사를 함께 보세요 — 둘 다 통과해야 신뢰할 수 있는 fitting입니다.

## 흔한 오류 Top 5

1. **금속을 대칭 피크로** → 산화물 과대평가
2. **doublet 비율 자유** → 비물리적 해
3. **성분 수 과잉** → χ²는 좋지만 화학적 의미 없음
4. **Shirley 끝점을 피크 위에** → 면적 왜곡
5. **전이금속 2p의 multiplet 무시** → Biesinger 권장 envelope 사용 (DB의 ref 참조)
