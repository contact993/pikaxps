# Corepeak 홈페이지 트래픽 늘리기 — 실행 방안

> 핵심 원칙(리서치 결론): 이 분야에서 트래픽은 **광고가 아니라 "논문 + 백링크 + 커뮤니티"**에서
> 옵니다. SEO는 보조. 아래는 효과 순으로 정렬했고, ✅는 이미 적용, ⬜는 사용자가 할 일.

## 이미 적용된 온사이트 SEO ✅
- ✅ schema.org `SoftwareApplication` JSON-LD (구글 리치결과 — 플랫폼·가격·다운로드 노출)
- ✅ 페이지별 title/description, `keywords` 메타, XML sitemap, robots.txt
- ✅ Open Graph + Twitter `summary_large_image` 메타 + 스크린샷 → 링크 공유 시 미리보기 카드
- ✅ 검색 타깃 키워드를 콘텐츠에 반영: *free XPS fitting software, XPS peak fitting software,
  XPSPEAK alternative / Mac, CasaXPS free alternative, XPS fitting software Mac*
- ✅ 롱테일 가이드(Shirley vs Tougaard, doublet, fit audit) — 정보성 검색 유입 + 다운로드 유도
- ✅ 한/영 이중언어 → 한국·아시아 검색 + 영어권 검색 동시 공략

## 1순위 — 가장 큰 트래픽 (off-site, 사용자)
- ⬜ **소프트웨어/방법론 논문 게재** (fit audit 앞세워, *Surf. Interface Anal.* 또는 *JVST A*).
  Google Scholar 색인 + 인용 = 가장 큰 지속 유입. KherveFitting도 22 star인데 논문이 유입원.
- ⬜ **Biesinger `xpsfitting.com` + HarwellXPS + `library4xps.org` 백링크 요청**.
  .edu/권위 사이트 백링크 1개가 신규 도메인 SEO를 몇 달 앞당김. (이미 Biesinger를 인용 중이므로
  "당신 레퍼런스 값을 도구화했다"로 프레이밍)
- ⬜ **ResearchGate "XPS 소프트웨어 추천?" 스레드에 저자 공개 답변** — 수년간 구글에 남아 유입.

## 2순위 — 검색·발견 인프라 (일부 사용자 계정 필요)
- ⬜ **Google Search Console 등록** → sitemap 제출 → 색인 요청 (도메인 연결 후). 신규 도메인 색인 가속.
- ⬜ **Bing Webmaster Tools**도 동일 (의외로 학계 트래픽 있음).
- ⬜ **도메인 연결**(corepeak.app 등) — `github.io/corepeak`보다 기억·신뢰·SEO 모두 유리.
- ⬜ **GitHub repo 토픽/About 정리** (xps, peak-fitting, spectroscopy, surface-analysis 태그) →
  GitHub 검색·토픽 페이지 유입.

## 3순위 — 콘텐츠·소셜 드립 (지속)
- ⬜ **YouTube 튜토리얼**(fit audit, doublet 피팅) — "how to fit XPS" 영상은 검색 잘 됨, 설명란에 링크.
- ⬜ **가이드 페이지 추가**(검색량 큰 롱테일): "XPS charge referencing", "C 1s fitting",
  "Fe 2p / Ni 2p fitting", "shake-up satellite" — 각 1페이지, 정보성 유입 → 다운로드 퍼널.
- ⬜ **LinkedIn / X**에 before-after fit·릴리스 노트 드립 (분석가·벤더 도달).
- ⬜ **한국 학회**(KVS/VASSCAA 제주 2026.8) 포스터 QR → 사이트.

## 광고(트래픽 자체보다 소액 수익용)
- ⬜ EthicalAds/Carbon 가입 후 docs 페이지 광고 슬롯 ID 채우기 (이미 코드 자리 있음). 트래픽이 커진 뒤 의미.

## 측정
- ⬜ 경량 애널리틱스(예: Cloudflare Web Analytics, 쿠키리스) 또는 GitHub Releases 다운로드 카운트로
  유입·다운로드 추적 → 그랜트/상근직 신청의 "broader impact" 근거로 누적.

(3) 도메인+Search Console. 이 셋이 광고·소셜보다 훨씬 크다.
