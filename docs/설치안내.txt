XPSFit Studio — 맥 설치 안내
================================

1. "XPSFit Studio.app.zip"을 다운로드한 뒤 더블클릭으로 압축을 풉니다.
2. "XPSFit Studio.app"을 응용 프로그램 폴더(또는 아무 곳)로 옮깁니다.
3. 첫 실행: 아이콘을 "우클릭 → 열기 → 열기"
   (개발자 서명이 등록되지 않은 앱이라 일반 더블클릭은 처음에 차단됩니다.
    한 번 열고 나면 이후엔 더블클릭으로 실행됩니다.)

※ "손상되었기 때문에 열 수 없습니다" 메시지가 뜨는 경우:
   터미널을 열고 아래 한 줄을 실행한 뒤 다시 우클릭 → 열기 하세요.
   xattr -cr "/Applications/XPSFit Studio.app"
   (앱을 다른 위치에 두었다면 경로를 그에 맞게 바꾸세요)

※ 지원 기종: Apple Silicon 맥 (M1 이후). Intel 맥에서 돌려야 하면 말해주세요.

--------------------------------
빠른 시작 (30초)

1. File → Import Data… 로 XPS 데이터(.dat/.txt/.csv/.xlsx/.xls) 열기
   - Thermo Avantage 엑셀 export(CHR-Ir.xls 같은 시트 구조)도 그대로 열립니다
     → Sheet에서 "Ir4f Scan" 등 선택만 하면 끝
   - 미리보기에서 데이터 시작 줄 클릭, X/Y 열 선택, KE→BE 변환 가능
   - Region 이름을 "Ni 2p"처럼 지으면 우측 DB가 자동으로 그 원소를 띄움
   - 또는 엑셀에서 두 열 복사 후 📋 Paste (⌘⇧V) 로 바로 붙여넣기
2. 우측 Reference DB에서 화학 상태 선택 → 가이드선 표시 / Doublet 삽입 / Recipe 삽입
3. ⚡ Fit Region
4. File → Export 로 파라미터 CSV·Excel·논문용 그림 저장

우측 "도움말" 탭에 백그라운드/라인섀입/constraint/대전보정/권장절차
한국어 설명이 들어 있습니다 (출처 포함).

※ 직접 참고하는 문헌값은 Reference DB 패널의 "＋ 내 레퍼런스"로 추가하세요.
   ~/.xpsfit/user_refdb.json 에 저장되며(★ 표시) 앱을 새로 받아도 유지됩니다.
   이 파일을 복사하면 다른 컴퓨터에도 같은 DB를 쓸 수 있습니다.
