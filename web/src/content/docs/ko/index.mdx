**Corepeak**은 익숙한 XPSPEAK 4.1 워크플로를 현대적 인터페이스로 재현하고, 한 걸음 더 나아갑니다.
단순히 XPS(X선 광전자 분광) 데이터를 피팅하게 해주는 게 아니라, **그 fit이 물리적·통계적으로
타당한지 검증**하고, 문헌 결합에너지를 알고 있어서 일일이 찾아볼 필요가 없습니다.
